# Currency Converter

Сервис реализовывает конвертирование пары валют.

## API

Релизован эндпоинт `GET /api/rates/` со следующими qгery-параметрами:
- from - тикер базовой валюты;
- to - тикер целевой валюты;
- value (опционально) - количество целевой валюты.


## Запуск сервиса

Для установки зависимостей необходимо:
- установить poetry;
- выполнить команду poetry install;

Запуск сервиса можно осуществить командой `poetry run uvicorn src.main:app` или в контейнере командой `docker compose up`.
